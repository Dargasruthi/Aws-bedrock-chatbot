# 📚 AWS Bedrock AI Chatbot

AI Knowledge Base Chatbot built using AWS Bedrock and Streamlit.

## 🚀 Features
- Ask questions from knowledge base
- AI-generated responses
- AWS Bedrock integration
- Streamlit UI

## 🛠 Tech Stack
- Python
- Streamlit
- AWS Bedrock
- Boto3

## ▶ Run Project
streamlit run app.py
